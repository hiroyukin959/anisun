
# アニメに関する話題をGitHubで！

仕事中の骨休みにいいかも。開発環境から確認できるのでいいね！

## ディレクトリ構成

### 第一階層
年_連番_クール

連番_クールのパターンは以下の４通り  
1_冬
2_春
3_夏
4_秋

例） `2022_2_春`

### 第二階層

アニメタイトル

例） `阿波連さんははかれない`
## README.mdの内容

### 第一階層

* そのクールについての感想
* タイトル一覧

### 第二階層

* そのアニメタイトル全体についての感想
* 各話一覧
* 各話に対する感想

## 感想の記入方法

どこにでも以下の形式で記載。

感想文(日付:Github Name)

例） なんかいい。(2022/04/02:hrn)
